What is Docker

Docker is an instrument that was designed to make it easier to run applications. This is done by using what is known as containers. Containers package up the work of developers and make it easier to view all together.

What is Nginx

Nginx is mainly an open-source web server, but has other functions. This webserver was designed for "maximum performance and stability." It has a simple configuration. Nginx also serves as a proxy server.

What is VM (virtual machine)

A virtual machine is a machine within a machine that essentially "emulates the computer system." It provides the same functions as a computer, but allows you to access hardware that may not be on the physical computer.

Who is Cloud Computing Service Provider - Digitalocean

Digitalocean is a cloud infrastructure provider that is based in New York. It provides a functioning platform for developers to work on their various projects. Digitalocean has become very popular, serving many different countries.
